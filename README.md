# Better Chromium

## What is Better Chromium?
Better Chromium is a project that applies patches to the Chromium source code to restore and improve functionality, such as adding Manifest Version 2 support back.

## Why Better Chromium?
Better Chromium gives a better experience to Chromium and re-adds removed functionality such as Manifest Version 2 support, allowing people to use older extensions and uBlock Origin.

## Features
- [x] Manifest Version 2 support
- [x] DuckDuckGo as default search engine
