# Better Chromium

## What is Better Chromium?
Better Chromium is a project that applies some patches to the Chromium source code that makes Chromium better, such as adding Manifest Version 2 support back.

## Why Better Chromium?
Better Chromium gives a better experience to Chromium and re-adds removed functionality such as Manifest Version 2 support, allowing people to use older extensions and uBlock Origin.
